## 1. What Are You Building? (2-3 sentences)

Describe your app like you're explaining it to a friend.

**Example:**
```
I want to build a focus timer app for people who struggle to concentrate. It will have
a 25-minute timer, a task list, and calming music. Like Pomodoro apps but with better
music and a cleaner design.
```

**Your turn:**
```
[Write your description here]
```

---

## 2. Who Is This For?

Who will use your app?

**Example:**
```
- Students studying for exams
- Freelancers working from home
- Anyone who gets distracted easily
```

**Your turn:**
```
[Who will use this?]
```

---

## 3. What Can Users DO? (5 Main Features)

List 5 things users can do in your app. Don't worry about HOW yet, just WHAT.

**Example:**
```
1. Start a 25-minute focus timer
2. Add tasks to a todo list
3. Play focus music while working
4. See how many sessions completed today
5. Track total focus time this week
```

**Your turn:**
```
1. [First thing users can do]
2. [Second thing]
3. [Third thing]
4. [Fourth thing]
5. [Fifth thing]
```

## 5. How Should It Look?

Describe the vibe you want, or name apps you want to copy the style from.

**Example:**
```
Clean and minimal like Linear. Dark mode like Spotify. Smooth animations like Apple.
```

**Your turn:**
```
[Describe the look/feel or name apps you like]
```

## 6. What Makes Your App Special?

What's the ONE thing that makes your app different or better?

**Example:**
```
Most focus apps play generic music. Mine uses AI-generated "flow state" music
specifically designed to help you concentrate without distraction.
```

**Your turn:**
```
[Your unique thing]
```

## 9. Will You Charge Money for It?
